### File and Directory Operations
| **Command**                 | **Description**                                                |
|-----------------------------|--------------------------------------------------------------|
| `cd [folder]`               | Change directory to the specified folder.                    |
| `cd ..`                     | Move to the parent directory.                                |
| `dir`                       | List files and folders in the current directory.             |
| `mkdir [folder]`            | Create a new folder.                                         |
| `rmdir [folder] /s /q`      | Delete a folder and its contents (`/s` = recursive, `/q` = quiet). |
| `del [filename]`            | Delete a specific file.                                      |
| `rename [oldname] [newname]`| Rename a file or folder.                                     |
| `copy [source] [destination]` | Copy files to another location.                           |
| `move [source] [destination]` | Move files to another location.                           |
| `attrib`                    | Change file attributes (e.g., read-only, hidden).           |

### System Information and Diagnostics
| **Command**               | **Description**                                              |
|---------------------------|------------------------------------------------------------|
| `ipconfig`                | Display network configuration details.                     |
| `ping [address]`          | Test network connectivity to a server or website.          |
| `netstat`                 | Display active network connections and ports.              |
| `tasklist`                | List all running processes on the system.                  |
| `taskkill /pid [pid] /f`  | Forcefully terminate a process using its process ID.        |
| `systeminfo`              | Display detailed system information.                       |
| `chkdsk`                  | Check disk for errors.                                     |
| `sfc /scannow`            | Scan and repair system files.                              |
